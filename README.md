# BCC Digital Forensics Sample Data  

This GitHub repository contains sample data originally collected or created for Cal Lee and Kam Woods’s SAA “Advanced Digital Forensics” class in 2021. The data can be used on its own for testing, teaching, and learning, or it can be used in conjunction with the [BitCurator Hands-On Exercises](https://bitcuratorconsortium.org/bitcuratoredu-learning-objects/#hands-on-exercises) compiled as part of the [BitCuratorEdu project](https://bitcuratorconsortium.org/bitcuratoredu/).



Use git to clone the repository to include all hosted files:

``` $ git clone https://github.com/BitCurator/bcc-dfa-sample-data.git```
