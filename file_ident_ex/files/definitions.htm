Year 2000 Status-related Definitions
Status Definition
User Tested & Accepted The Agency has repaired the system. The user has tested the system and signed off that the system has passed the required   tests.
Vendor Certified The Agency indicates that the Vendor has reported the system as Year 2000 ready and is certifying that the system will function properly.
3rd Party Verified The Agency reports that an independent consultant that did not develop the computer system has tested the system and has verified that the system is Year 2000 ready.
Repaired The asset has been repaired and tested by the repairer (but not by the User or a third party). Example: code has been modified to include four digit years, system software has been replaced with a Year 2000 "ready" version, PC BIOS upgraded.
Active Repair The asset is not Year 2000 ready but it is currently being repaired, or the replacement asset is being readied.
Hold The asset is being assessed as to whether it is in need of repair, or it has been determined to require repair or replacement, but activity has not been initiated.
 

Terms used on State Agency Status Pie Charts Only
Ready The Agency has reported that the IT Asset is Year 2000 ready because the asset was Vendor Certified, 3rd Party Verified, or User Tested and Accepted.
Status Not Reported The Agency has not provided a status on an Asset; therefore, we do not know the status.
